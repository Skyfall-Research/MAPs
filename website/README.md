# Skyfall UI - UI for the Skyfall Game

## How to Run:

1. [`npm`](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) must be installed.
2. Clone the repository
3. `npm i`
4. `npm run dev`
5. Go to [http://localhost:5173/](http://localhost:5173/)

## Tech Stack:

- [**SvelteKit**](https://svelte.dev/docs/kit/introduction): full stack framework.
- [**Tailwind**](http://tailwindcss.com/): CSS styling using HTML classes.
- [**ShadCN-Svelte**](https://shadcn-svelte.com/): UI component library.
- [**Tanstack table**](https://tanstack.com/table/latest): Library to manage data table (leaderboard) state.
- [**SvelteKit Superforms**](https://superforms.rocks/): SvelteKit form library.
- [**Zod**](https://zod.dev/): Schema validation library for use with Superforms.
- [**Lucide icons**](https://lucide.dev/): Icon library.
- [**three.js**](https://threejs.org/): 3D rendering library for the background animation.
- [**Vite**](https://vite.dev/): Build tool.

## Notes:

- The `src/lib/components/ui/` directory is automatically generated by ShadCN-Svelte. It contains all the UI component code. 
- Same goes for the `src/lib/hooks/` directory and the `src/lib/utils.ts` file.

## Limitations:

- The navbar breaks on mobile.
- When changing the difficulty on the leaderboard page, the table area shrinks or expands, and so the submission form moves, either up or down.
